소개

AI 기술을 통해 실질적인 가치를 창출하는 개발자가 되고자 합니다. 부트캠프 과정에서 RAG 기반 경제정보 챗봇 시스템을 설계하며 LLM의 실제 서비스 적용 과정을 경험했습니다. GPT Function Calling과 Vector Store를 활용한 문서 검색 시스템 구축, 실시간 API 통합, STT/TTS 음성 인터페이스 개발 등을 통해 AI 모델을 사용자 중심 서비스로 구현하는 방법을 익혔습니다. 단순히 모델을 학습시키는 것을 넘어, 사용자가 실제로 활용할 수 있는 AI 서비스를 만들고 싶습니다. 지속적인 학습과 실험을 통해 LLM 기술의 가능성을 탐구하며, 팀 협업을 통해 더 나은 솔루션을 만들어가는 개발자로 성장하겠습니다.

프로젝트 개요

경제 뉴스를 효율적으로 분석하고 핵심 이슈와 감성 흐름을 한눈에 파악할 수 있는 AI 기반 플랫폼입니다. GPT-5와 RAG 기술을 결합하여 실시간 경제 질의응답이 가능한 챗봇을 구현했습니다.

주요 담당 업무

  • 전체 시스템 아키텍처 설계 및 FastAPI 기반 백엔드 서버 구축
  • GPT-5 Function Calling을 활용한 AI 챗봇 개발 (실시간 경제지표 조회, RAG 기반 문서 검색)
  • MongoDB Atlas 연동으로 최신 뉴스 데이터 실시간 수집 및 관리
  • ECOS(한국은행), FRED(미국 연준), yFinance API 통합하여 경제지표 자동 조회 시스템 구현
  • CLOVA STT와 Google Cloud TTS 연동으로 음성 대화 기능 구현
  • Vector Store 기반 RAG 시스템 구축 및 자동 문서 갱신 파이프라인 개발
  • 팀 협업 관리 및 일정 조율, 최종 발표 준비
1단계: 데이터 수집
NAVER News API로 경제 뉴스 수집 · YouTube API로 영상 데이터 수집
ECOS API로 한국 경제지표 수집 · FRED API로 미국 경제지표 수집 · yFinance로 실시간 주가/환율 데이터 수집
↓
2단계: 데이터 저장
MongoDB Atlas (chatbot_rag 데이터베이스)에 수집된 모든 데이터 저장 및 관리
↓
3단계: AI 분석 처리
KoBART Summarizer로 뉴스 요약 생성 · Sentiment Analysis로 감성 분석 수행
GPT-5 Labeling으로 주요 키워드 및 카테고리 자동 분류
↓
4단계: AI 챗봇 엔진
GPT-5 Function Calling으로 사용자 질의 분석 및 함수 호출
Vector Store 기반 RAG로 관련 문서 검색 및 컨텍스트 제공
↓
5단계: 웹 서비스 통합
Spring Boot (8081) ↔ FastAPI (8002~8009) 간 RESTful API 통신
Thymeleaf 템플릿 엔진으로 동적 페이지 렌더링
↓
6단계: 사용자 인터페이스
실시간 대시보드로 경제 지표 시각화 · 뉴스 요약 자동 제공
AI 챗봇 대화 인터페이스 · Chart.js로 감성 분석 결과 그래프 표시

  • RAG 시스템: watcher.py를 통한 문서 자동 감시 및 Vector Store 실시간 갱신
  • API 엔드포인트: /chat, /tts, /stt, /rag/docs 등 RESTful API 설계
